# WorldExplorer_China-Scripting
WorldExplorer is a Python-based travel recommendation system focused on China. The project combines web scraping, APIs, data analysis, maps, and dashboards to help travelers discover destinations based on their interests and travel style.
WorldExplorer is a travel recommendation project developed in Python for the Scripting course project. The application focuses on China and helps travellers discover destinations and activities that match their personal travel preferences.

The project combines data from web scraping, external APIs, and data analysis to create a smart travel guide. Users can create their own traveller profile by selecting preferences such as budget, interests, travel period, and travel style. Based on this information, the system recommends destinations, attractions, and activities across different regions of China.

Besides recommendations, the project also includes:
- Interactive attraction maps
- Seasonal travel analysis
- Match score visualisations
- CSV/database exports for a Node-RED dashboard
- Travel tips generated through an LLM API

The goal of this project is to combine scripting, data processing, APIs, visualisation, and automation into one practical and interactive application.
